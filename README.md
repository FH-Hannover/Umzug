# UmzugPro – getrenntes Frontend/Backend (aktueller Stand)

## Frontend
- Start: `frontend/index.html`
- Enthält alle UI-Dateien, Assets, CSS, JS sowie optionale `partials/` für Teamarbeit.

## Backend
- `backend/admin/` enthält das Admin-Panel (UI), später vom Server geschützt ausgeliefert.
- `backend/data/` enthält eine Kopie der JSON-Dateien als Startpunkt für DB/API.
- `backend/src/` ist vorbereitet für API/Server.

Hinweis: Aktuell ist das Backend nur Struktur/Platzhalter – die Website funktioniert weiterhin statisch.
